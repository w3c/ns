ITS - XLIFF Namespace

This document describes the namespace for using Internationalization Tag Set (ITS) Version 1.0 and ITS 2.0 with XLIFF (Localization File Interchange Format) version 1.2 and 2.0. The namespace has been produced by the MultilingualWeb-LT working group and is maintained by the ITS Interest Group.

Please note that this namespace may change without notice. For updated information, please refer to the latest version of the Internationalization Tag Set (ITS) Version 1.0 and Internationalization Tag Set (ITS) Version 2.0 specifications, and to the MultilingualWeb-LT working group and ITS Interest Group pages.
