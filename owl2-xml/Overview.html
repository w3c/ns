XML Serialization Namespace for OWL 2

The namespace name http://www.w3.org/ns/owl2-xml is intended for use with the OWL 2 Web Ontology Language: XML Serialization. The following formal descriptions are available:

  • XML Schema 2009-09-21 (RECOMMENDED)
  • XML Schema from working group wiki (archived)

This document is intended to comply with W3C policy for XML namespace names.
