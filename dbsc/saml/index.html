Device Bound Session Credentials (DBSC) for SAML Namespace

This is the namespace for the Device Bound Session Credentials for SSO specification (SAML extension).

For further details, see the explainer: https://github.com/WICG/dbsc-sso.

The XML Schema for this namespace can be found at: dbsc-saml.xsd.
