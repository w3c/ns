WAI-ARIA Namespace

The namespace name http://www.w3.org/ns/wai-aria/ is intended for use with the Accessible Rich Internet Applications (WAI-ARIA). Formal descriptions are available from the WAI-ARIA Schemata page.

This document is intended to comply with W3C policy for XML namespace names.
