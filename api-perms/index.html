Namespace for permissions for device APIs access

The Device APIs and Policy Working Group defines in Permissions for Device API Access a method to build URIs that can be used as identifiers for the specified permissions.

These URIs are built by appending the name of the said permission to the base URI http://www.w3.org/ns/api-perms/.

This method and this URI are the results of a very early consensus of the group, and are likely to be modified in the future; anyone relying on them should get in touch with the Working Group to be notified when these changes occur.

As of September 2010, the following URIs are defined based on the specification:

  • http://www.w3.org/ns/api-perms/
